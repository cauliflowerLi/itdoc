# 介绍
本文档主要用来对前端开发中涉及的技术规范、公共组件和工具的API等做介绍说明。新开发的公共组件或者工具的说明文档除了上传私服npm仓库之外，相关的说明文档要同时更新发布到这里，以供开发人员学习查看。
